Instructions

Every event is found in the 'All Events' page.

Events which are finished/done are found in the finished Events page and events which are not Finished yet are found in the Upcoming Events page.

You can add event in the Add Event page.

While adding an event, you must compulsorily add Event Name, Faculty Co-ordinators/Convener/Event date/Publicity date, they are mandatory.

Event description/ Event comments/ Event department/ Even Committee are optional.

Mandatory level for Faculty-Coordinators is set upto 2 and for conveners upto 1. You can add upto maximum level (i.e 3/2).

While adding an event you must set a publishing date to 'today' as it sets to exact time you are adding the event.

Event posters can be seen in All Events but cannot be added by uploaded during add event by any normal user.

You can add event poster by going into admin page, for doing this you must go to Profile/ Go to Admin page.

Only Admin Users can perform this operation.

For admin account, you must click on admin signup dring signing up.

Only admins can delete databases/ Coloumns.

Only admins can delete other user accounts.
